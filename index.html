Pipeline

Five stages. One command.

01
Code Parser
AST + regex extracts hyperparameters, seed, split ratio from your training script.
02
Model Inspector
Loads the saved model and reads params, feature count, output type directly.
03
Metrics Loader
Picks up evaluation metrics from JSON or extracts them from code patterns.
04
Local LLM
Hermes3 via Ollama writes Intended Use and Limitations — grounded in real metadata.
05
Card Generator
Assembles a structured MODEL_CARD.md ready to commit to any repo.
Features

Everything a reviewer needs to trust your model.

EXTRACT
Zero-config extraction
Reads your existing code as-is. No decorators, no schema, no changes to your training pipeline.
LLM
Local LLM, real writing
Hermes3 runs on your machine. Your data never leaves. The prose it writes is grounded in actual extracted values.
SUPPORT
sklearn · XGBoost · PyTorch
Handles the three dominant ML frameworks. Detects model type automatically from imports.
CLI
Clean CLI interface
One command with sensible defaults. Override output path, metrics file, or LLM model as needed.
REPRO
Reproducibility section
Generates exact commands to retrain and regenerate the card — tied to your actual file names.
OUTPUT
GitHub-ready Markdown
Drop MODEL_CARD.md into any repo. Tables render on GitHub. No extra tooling required.
